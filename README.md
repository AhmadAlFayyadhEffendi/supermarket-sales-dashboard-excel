# Supermarket-Sales-Dashboard-Excel
This project is an interactive sales dashboard created using Microsoft Excel based on a supermarket sales dataset.
The dashboard is designed to help analyze sales performance and customer behavior through clear visualizations and dynamic filters

## Project Overview
The dashboard provides key business insights such as:
- Total sales
- Total products sold
- Average customer rating
- Monthly sales trends
- Sales by product line
- Rating by city
- Distribution of payment methods

It also includes interactive slicers to filter data by:
- City
- Customer type
- Product line
- Payment method
